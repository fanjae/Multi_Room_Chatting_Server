## 1. Project 소개 
- Multi Thread를 이용한 Chatting Room을 제공하는 서버이다.

## 2. 개발 환경
- **Server-side (C++)**
- **Client-side(C#)**
- **OS : Windows**

## 3. 서버 아키텍처
- 클라이언트-서버 모델
- 프로토콜 : TCP/IP
- 서버는 멀티스레드 방식으로 다중 클라이언트 요청 처리

## 4. 라이브러리 및 프레임워크
- C# (UI) : Windows Forms
- C++ : std::thread
